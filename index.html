Objetivo

O sistema simula uma central de atendimento de TI para registrar, consultar e controlar chamados técnicos.

Funcionalidades

Login, cadastro de usuários, cadastro de chamados, edição, exclusão, pesquisa, status e prioridades.

Desenvolvido com HTML, CSS, JavaScript e LocalStorage.

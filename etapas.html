Descrição das Etapas do Sistema

Este projeto simula o ambiente de uma central de atendimento técnico real, validando os requisitos funcionais e não-funcionais solicitados[cite: 5].

  • Etapa 1: Planejamento da arquitetura de páginas e definição de escopo (RFs e RNFs)[cite: 14, 20, 24].
  • Etapa 2: Estruturação semântica das telas utilizando HTML5 puro.
  • Etapa 3: Estilização visual moderna e responsiva com foco na usabilidade em dispositivos móveis utilizando CSS3[cite: 3, 11, 25].
  • Etapa 4: Implementação da lógica de controle (CRUD) e tratamento dos filtros dinâmicos de pesquisa com JavaScript Vanilla[cite: 5, 18, 21, 22].
  • Etapa 5: Integração com persistência persistente sem necessidade de banco de dados relacional via API Web LocalStorage[cite: 3, 18, 22].

Etapas do Projeto

Cada tópico foi separado em uma página para facilitar a apresentação da atividade.

1. Tela de Login

Permite autenticação de usuários, cadastro de novos acessos e proteção das páginas internas.

2. Dashboard

Exibe indicadores com total de chamados, chamados abertos, em andamento e finalizados.

3. Cadastro de Chamados

Permite registrar chamados com solicitante, setor, tipo do problema, prioridade, status e descrição.

4. Lista de Chamados

Permite consultar, pesquisar, editar e excluir chamados cadastrados.

5. Armazenamento Local

O sistema utiliza LocalStorage para salvar usuários, sessão de login e chamados no navegador.

Desenvolvido com HTML, CSS, JavaScript e LocalStorage.

# Sistema de Chamados de TI

Projeto acadêmico front-end com login funcional, cadastro de usuários e gerenciamento de chamados.

## Login inicial

Usuário: admin  
Senha: 123456

Também é possível cadastrar novos usuários na tela de login.

## Páginas

- login.html - Login e cadastro de usuários
- index.html - Dashboard
- novo-chamado.html - Cadastro de chamado
- chamados.html - Lista, pesquisa, edição e exclusão
- etapas.html - Etapas do projeto

## Funcionalidades

- Login funcional
- Cadastro de usuários
- Sessão simulada com LocalStorage
- Botão sair
- Proteção das páginas internas
- Cadastro, edição, exclusão e pesquisa de chamados
- Dashboard com indicadores

## Tecnologias

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Como executar

Abra o arquivo login.html no navegador.
